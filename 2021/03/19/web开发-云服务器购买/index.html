阿里云 服务器

背景

这篇文章主要在总结本次购买阿里云的经验，在这方面的知识我真的是一名小白，所以沉淀一下学习到的知识。

核心问题

核心问题包含三个部分：

  1. 选择ECS云服务器还是选轻量应用服务器
  2. CentOS,Ubuntu,Debian等不同的linux版本，到底怎么选
  3. 节点应该配置在哪个区域

1. ECS云服务器、轻量应用服务器如何选择

我遇到的第一个问题就是，轻量应用服务器、ECS(Elastic Cloud Service)应该怎么选择。
爬文之后，了解到，二者的区别在于：

  • ECS云服务器

    • 对用户的建站知识和技术有一定要求
    • 用户要按照个性化需要安装系统和软件，有系统管理权限，可弹性扩容

  • 轻量应用服务器

    • 轻量应用服务器面向新手提供一站式服务，把一键部署应用、一键域名解析、一键网站发布、安全、运维、应用管理等服务，通通集成在一个页面里面.
    • 主要场景：小型网站、个人博客、学习环境、小型电商网站、开发环境、论坛、管理工具、代码存储、代码测试等
    • 用于新手快捷上云推出的，与ECS服务器相比缺少内网
    • 特点是提供一键部署网站环境，以及一键搭建常见的网站程序。

简单来说，如果对linux语法不熟悉，对在服务器上安装nodejs应用等不熟悉，且所需量级较小，可以使用轻量应用服务器，快速完成部署。
但是对于需要定制化的需求来说，使用ECS云服务器更加合适。

2. CentOS, Ubuntu应该怎么选择？

CentOS

  • CentOS 是一个开源 Linux 发行版。
  • 许多人将其称为 Red Hat Enterprise Linux（RHEL）的副本，RHEL 被认为是企业 IT 领域中使用最广泛的软件。
  • 它具有高度可定制性，安全性和稳定性
  • CentOS 中，您必须使用 yum 命令从中央存储库下载并安装 RPM 软件包。
  • CentOS 被认为是更稳定的发行版。主要是因为软件包更新频率较低。
  • 如果您想使用控制面板来提供网络托管服务，那么您应该选择 CentOS。诸如 cPanel 之类的网络托管控制面板始终专注于 CentOS 和其他 RHEL 衍生产品。另一方面，Ubuntu 不支持 cPanel，但是有很多替代方案，例如Webmin / Virtualmin 或 VestaCP。
  • 企业主：CentOS 是您经营企业的理想选择，因为它的更新频率较低，因此（可以说）比 Ubuntu 更安全，更稳定。

https://cloud.tencent.com/developer/article/1763892

Ubuntu

  • 基于 Debian 架构的 Ubuntu
  • 会定期更新以删除任何已发现的错误
  • Ubuntu 中，您可以使用 apt-get 软件包管理器下载 DEB 软件包
  • 教程，在线社区和书籍方面，Ubuntu 当然占据了上风
  • 如果您是初学者，请使用 Ubuntu，因为它拥有更大的社区，更大的免费教程资源以及更频繁的更新，以帮助您保持最新状态。
